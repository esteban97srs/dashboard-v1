## Getting Started

Para correr la aplicación:

```bash
npm install (isntalación de depedencias necesarias)
npm run dev

Open [http://localhost:3000](http://localhost:3000) se corre en el puerto 3000 de igual manera en la terminal dice la URL para acceder.

## Técnologias usadas

- React js @latest
- Next js versión 12
- Maquetación guiada con ejemplo entregado en Figma
- React Query (consumo de datos, simulación de data utlizando archivos .json con la información)
- Routing (useRouter) Next Js para funcionalidad de ruteo
- Integración con Github Actions
- Implementación de Linter
- Proyecto estructurado en TypeScript
